⚔️ Download tool khai thác tại đây

Miêu tả

CVE-2023-22515 nói về lỗ hổng Broken Access Control trong Confluence Data Center and Server. Lỗ hổng này cho phép hacker có thể tạo tài khoản admin trái phép trên các hệ thống bị ảnh hưởng

Hệ thống confluence được xây dựng bằng Apache Struts. Và Apache Struts phụ thuộc vào Xwork – gói này cho phép định nghĩa các action dưới dạng một lớp java. Mỗi action được gọi thông qua url + lớp java tương ứng

Phiên bản ảnh hưởng

Tool khai thác

Dưới đây là tool mang mục đích kiểm tra server có phải là Atlassian confluence hay không. Nếu phải sẽ scan version server và thực hiện khai thác CVE nếu ở version bị ảnh hưởng.
